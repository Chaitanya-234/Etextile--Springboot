Etextile Website

I developed a full-stack e-commerce website for selling products using Spring Boot (backend) and Thymeleaf (frontend). The website includes features like user authentication, a shopping cart, multiple product categories, and Razorpay integration for handling payments. MongoDB is used for storing product, user, and order data.

I deployed the application on AWS using EC2 to run the Spring Boot JAR file, Route 53 to connect the custom domain purchased from Hostinger, and CloudFront to manage HTTPS requests. The website is secured with an SSL certificate generated using AWS Certificate Manager. For email notifications, I integrated AWS Simple Email Service (SES).

Technologies used: Spring Boot, Thymeleaf, MongoDB, AWS (EC2, Route 53, SES, CloudFront, Certificate Manager), Razorpay, Bootstrap.
